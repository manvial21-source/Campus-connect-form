# Campus-Connect Form

A clean, semantic, and well-structured Student Registration Form built entirely using raw HTML.

This project focuses on mastering the foundational concepts of web forms, emphasizing semantic HTML structure, accessibility, proper input handling, and organized form architecture before introducing CSS styling or JavaScript functionality.

---

## Features & Key Implementations

* **Semantic HTML Structure:**
  Utilized semantic elements such as `<header>`, `<main>`, `<footer>`, `<fieldset>`, and `<legend>` to create a meaningful and organized layout.

* **Accessible Form Design:**
  Implemented properly connected `<label>` and `<input>` elements using matching `for` and `id` attributes to improve accessibility and user interaction.

* **Diverse Form Controls:**
  Included multiple input types such as:

  * Text inputs
  * Email fields
  * Password fields
  * Telephone inputs (`tel`)
  * Radio buttons
  * Checkboxes
  * Dropdown menus
  * Textareas

* **Structured Form Organization:**
  Grouped related information into clearly separated sections for better readability and maintainability.

* **Beginner-Friendly Architecture:**
  Built entirely with pure HTML to strengthen understanding of core web development concepts without relying on CSS frameworks or libraries.
