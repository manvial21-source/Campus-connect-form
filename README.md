# Campus-Connect Form

A clean, semantic, and highly structured Student Registration Form built entirely with raw **HTML5**. 

This project was built to master the foundational structure of web forms, focusing heavily on accessibility, correct input types, and proper semantic element grouping before introducing CSS styling.


## 🛠️ Features & Key Implementations
* **Semantic Grouping:** Leveraged `<fieldset>` and `<legend>` tags to create distinct, readable sections for Personal, Address, Educational, and Additional information.
* **Accessible Inputs:** Used explicit `for` attributes on `<label>` elements perfectly matching the input `id` attributes to maximize form accessibility and hitboxes.
* **Form Controls:** Implemented diverse data inputs including text fields, radio buttons, phone types (`tel`), date pickers, and text areas.
* **Proper Form Architecture:** Wrapped the layout entirely in a structural `<form>` container using modern `<header>`, `<main>`, and `<footer>` semantics.
